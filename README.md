# Grundstückspreis-Rechner

Eine Python-Konsolenanwendung zur automatisierten Berechnung von Grundstückspreisen für Immobilienunternehmen. Das Programm verarbeitet Benutzereingaben zu Grundstücksmaßen und Quadratmeterpreisen, um den finalen Verkaufspreis inklusive Nebenkosten zu ermitteln.

## Projektbeschreibung (Aufgabe)
Entwicklung eines Algorithmus für den Immobilienhandel, der folgende Schritte ausführt:
1.  Einlesen von Länge, Breite und Quadratmeterpreis.
2.  Berechnung des Basis-Grundstückspreises.
3.  Aufschlag einer **Maklerprovision von 5%**.
4.  Berechnung der **Mehrwertsteuer von 19%** auf den Zwischenbetrag.
5.  Ausgabe aller relevanten Preisstufen formatiert auf zwei Nachkommastellen.
